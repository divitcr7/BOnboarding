# StackedMultiStepForm
This component comes with a custom hooks which can be used in any of subcomponents to manage state of stackMultiStepForm
<Canvas of={Stories.Example} />

# useStackedMultiStepForm
It can be used like

```js
const { onCompleteStep, onSkipStep, onBackStep, onNextStep, currentStep, totalSteps } = useStackedMultiStepForm();
```
